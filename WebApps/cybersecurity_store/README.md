## Execute

To run the original version of the web app:
```shell
cd app_org/
python -m venv venv
source venv/bin/activate
pip install cherrypy
pip install pillow
python main.py
```

To run the secure application:

```shell
cd app_sec/
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python main.py
```
